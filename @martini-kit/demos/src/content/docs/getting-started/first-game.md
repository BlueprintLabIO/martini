# Build Your First Game

You're about to build **Paddle Battle**, a complete 2-player Pong-style game. Follow the steps below to set up the project, wire up gameplay, and finish with polish and testing.

## Tutorial Steps

- [Part 1: Setup & State](/docs/latest/getting-started/first-game/01-setup) — create the project and define the shared game state.
- [Part 2: Gameplay Implementation](/docs/latest/getting-started/first-game/02-gameplay) — add paddles, ball physics, and networking.
- [Part 3: Finishing Touches](/docs/latest/getting-started/first-game/03-finishing) — add UI polish, persistence, and QA checks.

## What You’ll Need

- Node.js 18+ and your preferred package manager
- Familiarity with TypeScript and basic web dev tooling
- Optional: Phaser experience (the tutorial also shows core-only steps)
