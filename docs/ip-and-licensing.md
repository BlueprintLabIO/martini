## Remixes/Forks

### Remix Rights

When a game is published:
- Others can remix (create derivative works)
- Original creator is attributed
- Remix inherits license terms

**Remix attribution:**
```
Original game by @username
Remixed by @username2
View original: https://yourdomain.com/play/ABC123
```

### Opt-Out

Users can disable remixing:
- Setting: "Allow others to remix this game"
- Default: enabled for published games
- Private games: never remixable

## DMCA Policy

### Reporting Infringement

Email: dmca@yourdomain.com

Include:
- Your contact information
- Description of copyrighted work
- URL of infringing content
- Statement of good faith belief
- Statement under penalty of perjury
- Physical or electronic signature

### Response Process

1. Review claim (within 2 business days)
2. If valid, remove content immediately
3. Notify user of removal
4. User may file counter-notice
5. If no counter-notice in 10 days, permanent removal

### Repeat Infringers

- First offense: warning
- Second offense: temporary suspension (30 days)
- Third offense: permanent ban
