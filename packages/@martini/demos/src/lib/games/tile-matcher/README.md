# Tile Matcher

A turn-based competitive puzzle game for 2 players.

## Gameplay

- **Players**: 2
- **Type**: Turn-Based Puzzle
- **Controls**: Click to select tiles

## Objective

Match tiles to clear your board and score points. First to reach 500 points wins!

## Mechanics

- Take turns selecting and matching tiles
- Match 3 or more tiles of the same color
- Tiles fall down after matches (gravity)
- Chain reactions score bonus points

## How to Play

1. Host creates a room and shares the link
2. Second player joins
3. Players take turns matching tiles
4. First to 500 points wins

## Technical Details

- **Turn-based**: Only current player can make moves
- **Deterministic**: Same actions produce same results
- **State validation**: Host validates all moves
